## 技术特点

- 使用 ArkTS 开发，提供流畅的用户体验
- 支持多主题切换，适应不同用户偏好
- 集成多种功能，包括即时通讯、多媒体分享、小程序等
- 界面设计简洁直观，操作便捷

## 开发环境

- DevEco Studio
- HarmonyOS SDK
- 鸿蒙模拟器
## 如何运行
1. 克隆本仓库
2. 使用 DevEco Studio 打开项目
3. 配置必要的开发环境和依赖
4. 构建并运行项目
